# Tugas Praktikum 2

## Deskripsi
Ini adalah proyek sederhana yang menampilkan profil pribadi dalam bentuk halaman web. Halaman ini menggunakan HTML, CSS, dan JavaScript untuk membuat tampilan yang menarik dan interaktif.

## Fitur
- Menampilkan foto dan nama pengguna.
- Tombol untuk menampilkan popup deskripsi profil.
- Popup dapat ditutup dengan tombol close.

## Struktur File
- `index.html` : File utama yang berisi struktur halaman web.
- `style.css` : File CSS eksternal untuk styling halaman.
- `photo.jpeg` : Foto profil yang ditampilkan di halaman.

## Cara Menjalankan
1. Pastikan semua file berada dalam satu folder.
2. Buka `index.html` menggunakan browser.
3. Klik tombol "Lihat Detail" untuk menampilkan popup deskripsi.
4. Klik tanda "X" untuk menutup popup.

## Teknologi yang Digunakan
- **HTML** : Untuk struktur halaman.
- **CSS** : Untuk tampilan dan desain.
- **JavaScript** : Untuk interaksi popup.

## Catatan
Pastikan file `photo.jpeg` tersedia agar gambar profil dapat ditampilkan dengan benar.
